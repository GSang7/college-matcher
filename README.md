# 高考志愿智能匹配系统

基于性格测评与兴趣分析，为你推荐最适合的大学和专业。

## 功能特点

1. **MBTI性格测评** - 16型人格分析
2. **霍兰德职业兴趣测评** - 6种职业兴趣类型
3. **智能匹配算法** - 综合性格、兴趣、分数、偏好
4. **分档推荐** - 按分数段分层推荐
5. **院校详情** - 包含校园环境、食堂、宿舍评分

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

### 3. 访问应用

打开浏览器访问: http://localhost:5000

## 使用流程

1. **输入基本信息** - 高考分数、所在省份
2. **完成性格测评** - MBTI性格测试
3. **完成兴趣测评** - 霍兰德职业兴趣测试
4. **设置偏好** - 期望城市、环境要求
5. **查看推荐** - 获取个性化推荐结果

## 推荐算法

### MBTI性格匹配

根据16种性格类型，匹配适合的专业方向：
- INTJ: 计算机科学、数学、物理学、经济学等
- ENFP: 传播学、心理学、艺术学、市场营销等
- ... (共16种类型)

### 霍兰德兴趣匹配

根据6种职业兴趣类型，匹配专业：
- R(现实型): 机械工程、电子工程、土木工程等
- I(研究型): 物理学、数学、化学、生物学等
- A(艺术型): 艺术学、设计学、音乐学等
- S(社会型): 教育学、心理学、社会工作等
- E(企业型): 工商管理、市场营销、金融学等
- C(常规型): 会计学、审计学、统计学等

### 分数分档

- **顶尖高校** (680+): 985顶尖高校
- **重点高校** (620-680): 985及强势211
- **优质高校** (550-620): 普通211及强势双非
- **普通高校** (480-550): 普通本科院校
- **基础高校** (480以下): 普通本科及专科

## 数据说明

系统包含中国大陆主要高校数据，包括：
- 985/211/双一流高校
- 普通本科院校
- 各校开设专业
- 近五年录取分数线
- 校园环境评分
